开始学习Spring Boot啦，Let’s Roll 😉

前言

 老早就想学Spring Boot了，自己也试过，只是把官方demo启动起来，然后改改参数什么的，没有系统学习过，所以一起系统学习一下吧，文章中可能会有一些链接点不了，是以后会填的一些坑儿😂

简介

 Spring Boot简单来说就是Spring MVC的升级版，采用习惯优于配置的方式进行项目的快速部署开发，不必像之前需要进行各种的配置，同时它集成了大量常用的第三方库配置(例如Jackson, JDBC, Mongo, Redis, Mail等等)，而且还内置了Tomcat，真正达到了开箱即用的目的，让开发者更专注于业务。

系统环境

  • Java 1.8

  • Maven 3.6.1

    开发工具：IDEA 2019.1

第一个Spring Boot程序

1.访问https://start.spring.io/

2.选择构建工具、语言以及版本等信息

3.点击 Generate Project 下载项目压缩包

4.解压后使用IDEA导入项目，File>open…>选择项目中的pom.xml，别忘了将IDEA的Maven设置为自己下载的Maven，以便使用国内源，加快下载
